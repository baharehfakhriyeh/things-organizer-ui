# Things Organizer UI

**Things Organizer UI** is the frontend application for the **Things Organizer** API.  
It provides a GIS based interface to organize **Things** into **Containers**, manage **Contents**, and interact with the backend API.  
The project is built using **React 19**, **Vite**, and **Tailwind CSS**, with secure authentication handled via **Keycloak**.

---

## ✨ Features

- Display **Containers** on the map.
- Expand/collapse container details including subcontainers and things in a panel.
- Secure login and token refresh using **Keycloak**
- Responsive design powered by **Tailwind CSS**
- Real‑time updates with backend integration
